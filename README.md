## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/r8tten/Void.git
```

Move into the project directory:

```bash
cd Void
```

Install dependencies (if applicable):

```bash
# npm
npm install

# or
pip install -r requirements.txt

# or
cargo build

# Adjust depending on your project
```
